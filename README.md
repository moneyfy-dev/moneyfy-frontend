# MoneyFy

Mobile app built with Expo and React Native.

## Requirements
- Node.js >= 20.19.4
- npm

## Setup
1. Install dependencies
   ```bash
   npm install
   ```

2. Create your local env file
   ```bash
   copy .env.example .env
   ```

3. Update `API_URL` inside `.env`.

## Run
```bash
npx expo start
```

Open the app with Expo Go (SDK 54) or a development build.
